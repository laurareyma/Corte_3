# Diagramas del Sistema

Esta carpeta organiza los diagramas arquitectonicos por contexto:

- `src/`: vista de arquitectura general del sistema.
- `services/`: componentes y contenedores de microservicios.
- `client/`: interaccion cliente-servicios (secuencia).
- `monitoring/`: observabilidad y flujo de metricas/tracing.

Los diagramas estan en formato Mermaid (`.mmd`) para facilitar edicion y exportacion.
